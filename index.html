Project 3

This app builds on what we created in project 2. We are going to create a tic-tac-toe game that supports two game modes: pass-and-play and computer.

Requirements:
  • The start screen should have a label with the title of the game
  • The start screen should also have two buttons with the titles “Pass and Play” and “Play against Computer” that, when pressed, take you to a tic-tac-toe screen where you can play those game modes
  • When “Pass and Play” is clicked, the tic-tac-toe screen should work in the same way as described in the project 2 requirements
  • When “Play against Computer” is clicked, the tic-tac-toe screen should have the same appearance as in the “Pass and Play” mode
  • In “Play against Computer” mode, after the user plays a move, the computer should respond with a valid move
  • All of the content should be fully visible no matter the size of the screen
